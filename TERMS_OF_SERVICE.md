## Description of Service

The Application is an open-source tool that:
1. Monitors publicly available Reddit posts in subreddits where the Devvit sensor component is installed.
2. Identifies posts containing demand-intent signals using pattern matching.
3. Analyzes matched post text using AI-powered natural language processing.
4. Presents aggregated demand intelligence via a dashboard interface.
